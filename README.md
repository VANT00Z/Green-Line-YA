# Green Line

Молодёжный проект города Москва для упрощения процесса сдачи мусора на переработку.

## Описание

Green Line - это веб-сайт, созданный учениками 10 класса для упрощения процесса сдачи мусора на переработку. Здесь вы можете создать заявку на отправление ваших переробатываемых материалов.

## Технологии

- Django 5.1.5
- HTML5
- CSS3
- JavaScript

## Установка и запуск

### Требования

- Python 3.8+
- pip

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск проекта

1. Примените миграции (если необходимо):
```bash
python manage.py migrate
```

2. Запустите тестовый (локальный) сервер:
```bash
python manage.py runserver
```

3. Откройте в браузере:
```
http://127.0.0.1:8000/
```
